## React版我画你猜

之前有看到过一个Vue版本的 `我画你猜` 然后用 `React` 也做了一个。技术栈：`React + Nodejs + Socket.io + Webpack + Less`

第一次接触[Websocket](https://github.com/socketio/socket.io),主要注意`Socket.io`在`express`里的做法，具体可以参考官方。写完之后没有整理，比较乱。就是想大概实现一下，有兴趣的自己看吧！
